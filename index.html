Collection Manager

The Collection Manager website is built for everyone who loves to collect things but has a hard time remembering exactly what is in their collections. Using this web app, you can create and manage multiple collections with all of their included items. The collections can then be displayed publicly on your account or be kept privately for your viewing only.

On the frontend, the application features a responsive, user-friendly interface built with Next.js and React, using a multi-page architecture to provide intuitive navigation. The backend is developed with Node.js and is powered by a PostgreSQL database running in a docker container for efficient deployment and scalability.

Dallas Police Department Incident Report

The Dallas Police Department Incident Report is a data science dashboard built during my business analytics graduate program. The website contains the 2023 Dallas PD Incident report dataset and allows the user to view, create, modify, and delete any incident reports. The user can also view a heat map of the city displaying the reported incidents.

The frontend for this application was developed using the R Shiny framework and has full CRUD capabilities. It is connected to a SQL Server database to store all of the incident reports.
